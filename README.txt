Synth Strings #1
----------------

Version 2020-05-28

General MIDI instrument #51: Synth Strings 1. Made for the FreePats project, using ZynAddSubFX / Yoshimi software synthesizers. ZynAddSubFX and Yoshimi are free software programs capable of creating many different sounds with several synthesis engines, filters and effects.

Please visit FreePats web pages for updates and more content:
http://freepats.zenvoid.org/
